# VOICEVOX向け音声台本へ調整する
`video/production/script.md` を読み、`video/production/voicevox-script.md` を作成してください。

各台詞に次を持たせてください:
- scene_id
- speaker
- text
- reading
- emotion
- speed_hint
- pause_after_ms
- estimated_seconds
- subtitle
- display_only_text

内容や技術的主張は変えず、読み上げにくい表現を調整してください。
完成したら STEP_COMPLETE を出してください。
