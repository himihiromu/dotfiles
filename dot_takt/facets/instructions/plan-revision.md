# 修正計画策定
`video/production/review-summary.md` と既存の成果物を確認し、修正計画を作成してください。

`video/production/revision-plan.md` に以下を記載してください。
- 修正対象シーンと理由
- 各修正の影響範囲（blast radius）
- 修正の実行順序
- ストーリーボードへの影響
- Remotion実装への影響
- 音声再録音の必要性
- 字幕変更の有無

修正は最小限の影響範囲にとどめ、壊れていない部分はそのまま保持してください。

結果は出力契約に従い、最後に STEP_COMPLETE / STEP_BLOCKED のいずれか一つを出してください。
