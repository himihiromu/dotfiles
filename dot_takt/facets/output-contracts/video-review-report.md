# Video Review Report
次の構造で報告してください。

## Review Scope
確認した成果物と観点。

## Findings
各指摘に以下を含める。
- Severity: Critical / Major / Minor / Suggestion
- Location: scene_id、台詞、見出し等
- Problem
- Impact
- Required Change

## Positive Findings
維持すべき良い点。

## Summary
Critical、Major、Minor、Suggestionの件数。

## Status
最後の行に、次のいずれか一つだけを記載する。
REVIEW_APPROVED
REVIEW_NEEDS_FIX
REVIEW_BLOCKED
