# Revision Plan
次の構造で報告してください。

## Scope
修正の全体範囲と方針。

## Target Scenes
修正対象の各シーンについて以下を記載。
- Scene ID / 該当箇所
- Reason: 修正理由
- Impact: 影響範囲
- Order: 実行順序
- Change Description: 具体的な変更内容

## Storyboard Impact
ストーリーボードへの影響。変更が必要なシーンと演出。

## Remotion Impact
Remotion実装への影響。コード変更が必要な箇所。

## Re-recording Required
音声の再録音が必要な変更のリスト。なければ「なし」。

## Subtitle Changes
字幕の変更が必要な箇所。なければ「なし」。

## Status
最後の行に、次のいずれか一つだけを記載する。
STEP_COMPLETE
STEP_BLOCKED
