# Content Analysis Report
次の構造で報告してください。

## Overview
分析対象の動画と概要。

## Duration Analysis
再生時間の分析。目標尺との比較、各パートの時間配分。

## Tempo
シーンごとのテンポ分析。展開が速すぎる・遅すぎる箇所。

## Scene Structure
導入・本編・結末の構成比率と適切性。

## Conversation Ratio
解説文とセリフの比率。バランスの評価。

## Code/Demo Time
コード提示・デモの時間と頻度。視聴者の理解負荷。

## Subtitle/Info Density
字幕数と情報密度。画面の情報過多または不足。

## Audience Alignment
視聴者層（初心者〜上級者）との整合性。

## Findings
改善すべき点と維持すべき点のリスト。
