# Producer Review Report
次の構造で報告してください。

## Release Assessment
公開可能性と全体所見。

## Blocking Findings
公開を妨げる問題。なければ「なし」。

## Non-blocking Findings
任意改善。

## Revision Owner
script / storyboard / implementation / external-input / none のいずれか。

## Completion Criteria
次回レビューで満たすべき条件。

## Final Decision
最後の行に、次のいずれか一つだけを記載する。
PRODUCER_APPROVE
PRODUCER_REVISE_SCRIPT
PRODUCER_REVISE_STORYBOARD
PRODUCER_REVISE_IMPLEMENTATION
PRODUCER_BLOCKED
